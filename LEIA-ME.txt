COMO APLICAR

1. Pare o servidor com Ctrl + C.
2. Extraia este ZIP.
3. Copie as pastas app e components.
4. Cole dentro de:
   DD_Prime_Site_Completo_V1\ddprime-site-v1
5. Escolha "Substituir os arquivos no destino".
6. No PowerShell, entre em ddprime-site-v1.
7. Execute:
   npm.cmd run build
8. Publique:
   git add .
   git commit -m "Corrige layout desktop e celular"
   git push origin main
