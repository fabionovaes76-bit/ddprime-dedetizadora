PUBLICAÇÃO
1. Substitua os arquivos do seu projeto pelos arquivos deste pacote.
2. Execute: npm.cmd run build
3. Execute: git add .
4. Execute: git commit -m "SEO completo e conversoes GA4"
5. Execute: git push origin main

GOOGLE ANALYTICS
Depois que os eventos aparecerem no GA4, marque generate_lead, click_whatsapp e click_phone como eventos principais conforme sua estratégia. Para Google Ads, importe os eventos principais pelo vínculo Analytics > Google Ads.

OBSERVAÇÃO
Não foi adicionado Review/AggregateRating Schema porque estrelas nos resultados só devem usar avaliações reais, elegíveis e compatíveis com as políticas do Google.
