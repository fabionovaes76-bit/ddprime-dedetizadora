- Nova logo e escudo premium
- Banner sem foto de técnico
- Serviços
- Diferenciais
- Como funciona

- SEO local, sitemap e robots.txt

## Publicação

Copie todos os arquivos desta pasta para `ddprime-site-v1`, substituindo os atuais.

Depois execute:

```powershell
npm.cmd install
npm.cmd run build
git add .
git commit -m "Publica site V8"
git push
```
