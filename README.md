## Publicar

Copie todos os arquivos desta pasta para `ddprime-site-v1`, substituindo os atuais.

Depois execute:

```powershell
npm.cmd install
npm.cmd run build
git add .
git commit -m "Publica versão definitiva do site"
git push
```
